
# Pet Adoption Website
 
 A ReactJs project bundled via Vite.

### Run Locally

Clone the project

```bash
  git clone https://github.com/NigamDevansh/BloggingSite_WithAuth
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
